## Ações do Sistema

### Chamado

- **Abrir Chamado** — Criar novo ticket
- **Alterar Status** — Mover no workflow
- **Alterar Prioridade** — Definir urgência
- **Listar Chamados** — Ver todos os tickets

- **Adicionar Comentário** — Responder chamado
- **Listar Comentários** — Ver histórico
